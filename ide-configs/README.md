# Configuraciones de IDEs

Esta carpeta contiene configuraciones predefinidas para diferentes IDEs que facilitan el desarrollo con Checkstyle.

## 📁 Estructura

```
ide-configs/
├── vscode/                    # Visual Studio Code
│   ├── settings.json          # Configuración principal
│   └── README.md              # Instrucciones de instalación
├── spring-tool-suite/         # Spring Tool Suite
│   ├── .checkstyle            # Configuración Checkstyle
│   ├── .project               # Configuración del proyecto
│   └── README.md              # Instrucciones de instalación
├── eclipse/                   # Eclipse IDE
│   ├── .checkstyle            # Configuración Checkstyle
│   └── README.md              # Instrucciones de instalación
├── intellij/                  # IntelliJ IDEA
│   ├── checkstyle-idea.xml    # Configuración Checkstyle
│   └── README.md              # Instrucciones de instalación
└── README.md                  # Este archivo
```

## 🚀 Uso

1. **Selecciona tu IDE** de la carpeta correspondiente
2. **Copia los archivos** a tu proyecto según las instrucciones
3. **Sigue el README** específico de cada IDE
4. **Reinicia tu IDE** para aplicar cambios

## 📋 IDEs Soportados

- ✅ **Visual Studio Code** - Configuración completa con plugins
- ✅ **Spring Tool Suite** - Configuración nativa de Checkstyle
- ✅ **Eclipse** - Configuración básica de Checkstyle
- ✅ **IntelliJ IDEA** - Plugin Checkstyle configurado

## 🔄 Actualización

Estos archivos se mantienen actualizados con las mejores prácticas y configuraciones del proyecto.
